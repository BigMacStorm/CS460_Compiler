1. Open directory
2. "make clean"
3. "make"
4. "./compiler [-d lsp] program.c < program.c"

Where "program" is the name of the program to run the compiler on.

Testcase programs are in the same directory, and are named
    "test1.c" ... "test6.c".

An output file of the same name but with extension ".list" will be created when the compiler is run on a ".c" program.

Log files will be created when the -d l, -d s, and -d p options are used.
